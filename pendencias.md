# Pendências do Projeto - Amazonia Travel e Events

Lista de tarefas e configurações pendentes para a finalização do projeto.

## Configurações Externas
- [ ] **Configurar domínio no Resend:** Para enviar emails a partir de `@amazoniatraveleevents.com`, é necessário verificar o domínio no painel do [Resend](https://resend.com/domains).
- [ ] **Configurar Secrets no Supabase:** Adicionar `RESEND_API_KEY` nas Edge Functions do Supabase.
